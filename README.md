# TecPonto

Site institucional e landing pages de conversao da TecPonto.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Rodar local

```sh
npm install
npm run dev
```

O servidor local usa a porta `8080`.

## Scripts

```sh
npm run dev
npm run build
npm run lint
npm run preview
```

## Estrutura principal

- `src/app`: shell da aplicacao e rotas
- `src/layouts`: header e footer globais
- `src/features`: funcionalidades por dominio
- `src/sections`: secoes reutilizadas nas paginas
- `src/pages`: paginas roteadas
- `src/assets`: imagens, videos, icones e materiais visuais
